# Parser — Node.js парсер авиабилетов

## Что делает
- Запускает браузер Playwright со stealth-плагином
- Парсит цены с сайта авиабилетов
- Сериализует данные в Protobuf
- Сохраняет в MongoDB с TTL-индексом

## Установка
```bash
npm install
```

## Запуск
```bash
npm start
```

## Для собеседования — ключевые моменты

**Почему Playwright?**
- Быстрее Selenium, современный API
- Поддерживает все браузеры
- Хорошая работа с динамическим контентом

**Почему stealth-плагин?**
- Скрывает признаки автоматизации
- Имитирует поведение реального пользователя
- Помогает обойти простую защиту от ботов

**Почему TTL-индекс?**
- Автоматически удаляет старые данные (30 дней)
- Не нужно писать отдельный скрипт для очистки
- База не разрастается бесконечно

**Обработка ошибок**
- Если капча или сайт недоступен — возвращаем демо-данные
- Graceful degradation — система продолжает работать
