# Analytics — Python аналитика

## Что делает
- Загружает данные из MongoDB
- Считает базовую статистику (средняя, мин, макс цена)
- Находит лучшее время для покупки
- Строит график цен

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python analytics.py
```

## Для собеседования — ключевые моменты

**Почему Python для аналитики?**
- pandas — удобная работа с табличными данными
- matplotlib — быстрая визуализация
- Большая экосистема для ML (scikit-learn, tensorflow)

**Почему базовая статистика?**
- Показывает, что умею работать с данными
- Легко объяснить на собеседовании
- Можно расширить до ML (линейная регрессия для прогноза)

**Обработка пустых данных**
- Проверка на None/empty перед обработкой
- Graceful degradation — не падаем, если данных нет
